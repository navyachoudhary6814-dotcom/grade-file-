# 🎓 Gradify

Gradify is a student productivity web application designed to help students manage their academic life in one place.  
It provides features like task planning, attendance tracking, notes management, and progress monitoring with a clean and modern interface.

---

## ✨ Features

### 📅 Planner
- Interactive calendar
- Add and delete daily tasks
- Task reminders
- Local storage support

### 📝 Notes Section
- Store and organize study notes
- Easy access to academic material

### 📊 Progress Tracker
- Track academic progress visually
- Monitor performance and productivity

### 📌 Attendance Management
- Calculate and manage attendance percentage
- Keep track of classes attended

### 🎨 Modern UI
- Responsive layout
- Clean dashboard design
- User-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
